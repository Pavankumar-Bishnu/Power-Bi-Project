# VEHICAL LOAN or CREDIT DATA ANALYSIS ( Finance )
Interactive dashboard creation - using Power BI 
> **Data Objective** ( Purpose of the data ) :

This dataset (or) project provides a comprehensive overview of potential loan applicants' profiles, specifically tailored for the Indian demographic. It encapsulates a range of features, from basic demographics to financial details, that can be instrumental in assessing the creditworthiness of an individual.

> **Dataset Used :** <a href="https://github.com/Pavankumar-Bishnu/Power-Bi-Project/blob/main/Credit_Data.csv"> Click Now </a>

  - Total Records            : 179,042
  - Total Features (Columns) : 15
  - Numerical                : 9
  - Categorical              : 6
  - Demographics             : Age, Gender, Occupation, Employment Profile, Income
  - Credit Information       : Credit Score, Credit History Length, Number of Existing Loans
  - Loan Information         : Loan Amount, Loan Tenure, LTV Ratio, Existing Customer
  - Location                 : State, City
  - Profile Metrics          : Profile Score

## **Final Conclusion:**
- Credit Score Distribution   : Most customers fall between 450–700, indicating an average-to-moderate risk borrower pool.
- Income vs Loan Amount       : Strong clustering shows that higher income tends to align with higher loan amounts, but with a cap near ₹1.5 Lakh.
- Gender Distribution         : Loans are more balanced between Male and Female, with a small portion as Other.
- Top Loan States             : West Bengal, Karnataka, and Uttar Pradesh contribute the most loan customers.
- Employment Profile          : Salaried individuals dominate, followed by Self-Employed and Business profiles.
